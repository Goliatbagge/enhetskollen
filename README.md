# Fysik-kollen

En interaktiv quiz-app för att träna på fysikaliska storheter, beteckningar och SI-enheter.

## Funktioner

- **Läge 1**: Storhet → SI-enhet (t.ex. "Hastighet" → "m/s")
- **Läge 2**: Beteckning → Storhet (t.ex. "v" → "Hastighet")
- Progressbaserad träning med direktfeedback
- Animerad "hjul"-design för frågor
- Mobilanpassad och responsiv

## Teknologier

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (animationer)
- Lucide React (ikoner)
- Vite (byggverktyg)
